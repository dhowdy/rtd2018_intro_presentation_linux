What is a Raspberry Pi?

An inexpensive credit card-sized, single board computer (SBC)
Runs a variety of operating systems (OSes), but typically Linux
Capable of hardware-accelerated full HD 1080P video output
40 or 44 General Purpose Input/Output (GPIO) pins to connect
Has additional accessories like a camera and touchscreen
Massive community following with 20,000,000 devices sold!!

Why the Raspberry Pi?

  • Longstanding goal to create a inexpensive computers for education
  • Created February 2012 by the Raspberry Pi Foundation in UK
  • Teaching basic computer skills to children
  • Named after a fruit (as many microcontrollers are) and Python
  • Substantial collaboration between the Foundation and Broadcom
  • Eben Upton, Broadcom, and the Foundation
  • Over 3 dozen operating systems available for the Pi
  • Build your own!!

What is an ESP8266?

An inexpensive WiFi microchip built for IoT
Designed to connect to microcontrollers to add WiFi capabilities
Came onto the scene in August 2014 with sketchy start
Had no English documentation or command interpreter
Incredibly fast community adoption and development

Why the ESP8266?

WiFi is ubiquitous. The ESP8266 initially cost $5.
ESP is cheap, durable, compact, easy on power
Microcontrollers like Arduino are very capable and...
ESP8266 promised to add WiFi to any microcontroller
Hackers soon learned that the ESP was far more capable on it's own
Went from a WiFi adapter to a standalone Wifi system overnight
There's now a dozen SDKs available for the ESP8266

Supported Hardware & Software Protocols

  • Serial/UART: Universal Asynchronous Receiver/Transmitter 1:1 device ratio, slow, easy in hardware, hard in software.
  • SPI: Serial Peripheral Interface; synchronous, 1 master to multiple slaves, faster than UART, high data rate, short distances
  • I²C: Inter-Integrated Circuit; 2 wire serial, simple multiple masters & up to 1008 slaves, high data rate, short distances
  • 1-Wire: Similar to I²C, lower data rate & longer distances
  • PWM: Pulse-Width Modulation; Digital pulse wave to approximate an analog signal. Changes the duration of the pulse to get desired waveform. Done in software.

Other features of the Raspberry Pi

  • One-Time Programmable Memory (OTP) can never be unset
  • USB Boot - Boot the Pi from USB mass storage device (3B/+)
  • PXE Boot - Requires DHCP, DNS, TFTP, and NFS servers (3B/+)
  • Also possible to boot from a second SD card, NAND flash, & SPI
  • Hack workarounds for USB and PXE booting on older models
